
# Development
```
pip install -r requirements.txt
cd webserver
python3 server.py
```
#### Optional
```
python3 -m venv path/to/venv  
source path/to/venv/bin/activate
```
#### TailwindCSS
```
npx tailwindcss -i ./static/css/style.css -o ./static/css/output.css --watch
```
